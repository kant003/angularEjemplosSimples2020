Incluyendo boostrap y angular material

Para instalar boostrap

npm install boostrap jquery popper

Color el siguiente código en el fichero angular.json

"styles": [ "src/styles.css", "./node_modules/bootstrap/dist/css/bootstrap.css" ], "scripts": [ "./node_modules/jquery/dist/jquery.js", "./node_modules/bootstrap/dist/js/bootstrap.js" ]

Otra forma de instalar boostrap es colocar el siguietne código(CDN) en index.html

Para instalar angular material instala esta dependencia

ng add @angular/material
